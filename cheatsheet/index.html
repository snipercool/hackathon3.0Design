Cheatsheet:

Get/make repository

Open your repository in code editor

Commit with summary and description

Git push

Pushes all the files you commited to your repository on github.

Git pull

If a teammate pushed code on the repository you have to pull the newly changed code to your files.

Git branch

Shows all the branshed in the repository.

Git branch <name>

Creats a branch in the repository.

Git branch -d <name>

Deletes a branch in the repository.

Git branch checkout <name>

Switch between branches in your repository.

Example: You are on branch master with git branch checkout login you switch to the branch login.

Git merge

merges the diffrences between 2 branches.

Example: You are on branch login before merging with master merge master with login on your branch make sure everything works and afterwards merge login with master.
